Game Information:

Produced by Alan Rominger, 2015, strictly for learning purposes. All images were made in Paint from scratch. Reuse whatever you want.

Philosophy:

The tiles of this game are created in a table, intentionally avoiding a canvas element, so all the interactions are coded to the page elements individually. That means you should experience some slowdown when using the "hardest" mode.
