Alan, Iron Yard, Cohort 4

This site is designed to showcase the Django projects from the Iron Yard in Durham in the Python class during summer 2015. There are 4 main apps that were produced, and these are combined here into a single project with a single user account system.

Movie Ratings

You can make an account on this site to rate movies, but it also has data taken from the MovieLens project pre-loaded into it, including 1 million reviews.

Go to Movies App »

Github Movies Repo »

URLy Bird

This is a URL shortener that creates a hash when a user submits a link. It also counts the clicks on the link and gives stats for different users.

Go to URLy Bird »

Github URLy Bird »

Questions Box

Clone of Stack Overflow. This was a group project with 2 people, and also one of my best products from the class. It gives a single-page view of a question that allows comments on the question and on the answers.

After finishing the Iron Yard course, I thought it would be nice to consolidate all the Django apps we made into a single app and get a nice-looking domain name for it as well. The only additional challenge is making a user account system that is consistent accross the different apps as well. There are some other projects from the code school I would like to add as well, but I don't yet have all the materials for these together.

Clickinator

This was a morning project done in a group, but we only had about an hour to work on it... and we needed to learn the tools while we were at it. The API calls are very central to my own personal learning goals so I hope to come back and polish it up soon.

Stats Tracker

Also a group project, this is a single-page app designed to let users mark the number of instances of something for a given date.
